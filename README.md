# 🌍 Trip Adviser

A simple and responsive travel website built using HTML and CSS. The website helps users explore travel destinations and discover exciting places around the world through a clean and attractive design.

## ✨ Features

- Responsive design
- Modern travel-themed layout
- Popular destination sections
- Attractive cards and images
- User-friendly navigation
- Mobile-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```text
Trip-Adviser/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## 🚀 How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/trip-adviser.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
